วาง Token ครั้งเดียว ระบบจะจำไว้ในเครื่องนี้ — ใช้แก้เมนู เปลี่ยนราคา ติ๊กของหมด และอัปโหลดรูปอาหาร

วิธีสร้าง Token (ทำครั้งเดียว ~2 นาที)
  1. เข้า github.com (บัญชี jitiwatjc) → รูปโปรไฟล์ → Settings
  2. ล่างสุด Developer settings → Personal access tokens → Fine-grained tokens → Generate new token
  3. ตั้งชื่อ kyoto-menu · Expiration เลือก 1 year
  4. Repository access → Only select repositories → เลือก kyoto-shi-cafe
  5. Permissions → Repository permissions → Contents → Read and write
  6. กด Generate token → คัดลอกโค้ดที่ขึ้นต้น github_pat_ มาวางที่นี่
⚠️ Token เก็บอยู่ในเครื่องนี้เท่านั้น ห้ามส่งต่อให้ผู้อื่น · ถ้าโทรศัพท์หาย ให้เข้า GitHub ไป Revoke token ทิ้ง
